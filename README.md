# EHEditor简介

[EHEditor](https://editor.benhuisen.com/) 作为一款功能强大的编辑工具，聚焦于文档编辑领域，致力于全方位提升用户编写体验，从编写规范、效率提升，渲染效果丰富以及交互性增强等多个维度，来满足不同用户多样化的记录需求。该编辑器同时支持**Spralang语法、Markdown语法、HTML语法，并支持公式、流程图等，可用来记笔记、写论文**等。

## 现有编辑器存在的问题

1. **视觉呈现受限**：传统文档编辑器（如WORD）、在线编辑器（如讯飞文档）在视觉呈现方面存在明显局限。例如，这些编辑器无法实现图片轮播、表格分页、自定义锚点样式，无法创建响应式布局等。

2. **缺乏交互性**：传统文档编辑器（如WPS）、文档语言（如Markdown）、在线编辑器（如有道云笔记）普遍缺乏交互能力，如无法做到表格翻页/搜索、手风琴展开/折叠、GIF图播放/暂停等。

3. **数据迁移受限**：Markdown表格数据包含较多管道符（|）和连字符（-），HTML表格则有大量`<tr>`和`<td>`标签，相比CSV格式数据存在一定冗余，且在与WPS等传统文档编辑器进行数据迁移时存在一定困难。

4. **编写效率低下**：现有Web技术（HTML/CSS/JS）已发展成熟，当下互联网环境中存在很多优秀的JS/CSS插件资源，但在利用这些技术进行文档编写时，却面临着复杂度较高的问题，这使得普通用户或非专业开发人员在利用Web技术进行文档编写时存在较大困难。

## 解决方案

为了解决以上问题，我们开发出了一种新型的文档编写语言（Spralang：/sprə-lɑːŋ/），该语言追求极致精简，仅通过一个参数指令，即可实现特定的渲染以及交互效果。以下列出部分示例，更多示例可参考`教程 > Spralang`。

### 本编辑器优势

1. **极大提高编写效率**：使用Spralang语言编写文档，通过指定标签属性即可实现特定的交互以及渲染效果，并可节省约86.7~95.7%的代码量，从而极大提高文档的编写效率。

2. **解决渲染交互受限问题**：示例3以及示例5表明，本编辑器可解决传统编辑器（如Word）以及在线编辑器（如讯飞文档）渲染及交互受限问题。

3. **管理方便**：支持多层级目录创建、拖拽文档/目录等，方便文档管理。

4. **多语言支持**：支持Spralang语法、Markdown语法、HTML语法、公式语法、流程图语法等。

5. **多种编辑/预览模式**：具体包括编辑模式、编辑+预览模式、预览模式等。

5. **支持主流前端框架**：内置Bootstrap5、Layui等主流框架，可规范文档编写，使文档具有具有现代化观感。

7. **提示/记忆功能**：本编辑器支持提示/记忆功能，可进一步提高文档编写效率。
